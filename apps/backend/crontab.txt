# Cron Schedule Format:
# * * * * * command
# │ │ │ │ │
# │ │ │ │ └─── Day of week (0-7, Sunday = 0 or 7)
# │ │ │ └───── Month (1-12)
# │ │ └─────── Day of month (1-31)
# │ └───────── Hour (0-23)
# └─────────── Minute (0-59)

# Examples:
# 0 0 * * * - Every day at midnight
# 0 */6 * * * - Every 6 hours
# 0 9 * * 1-5 - Every weekday at 9 AM
