## Getting Started

### Prerequisites

- Node.js 18+
- Python 3.11+
- iOS Simulator (for mobile development)
- Supabase account
- API keys for AI services

### Environment Setup

1. **Clone the repository**
```bash
git clone https://github.com/Zchasse63/voice-fit.git
cd voice-fit
```

2. **Copy environment variables**
```bash
cp .env.example .env
```

3. **Configure environment variables**
```env
# Supabase
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=your-anon-key
SUPABASE_SERVICE_KEY=your-service-role-key
SUPABASE_JWT_SECRET=your-jwt-secret

# AI Services
XAI_API_KEY=your-xai-key
KIMI_API_KEY=your-kimi-key

# Upstash
UPSTASH_SEARCH_REST_URL=your-search-url
UPSTASH_SEARCH_REST_TOKEN=your-search-token
UPSTASH_REDIS_REST_URL=your-redis-url
UPSTASH_REDIS_REST_TOKEN=your-redis-token

# Wearables (optional)
TERRA_API_KEY=your-terra-key
WHOOP_CLIENT_ID=your-whoop-id
WHOOP_CLIENT_SECRET=your-whoop-secret

# Auth
REQUIRE_AUTH=true
```

### Running the Mobile App

```bash
# Install dependencies
cd apps/mobile
npm install

# Start Expo
npm start

# Run on iOS Simulator
npm run ios

# Run on Web (for testing)
npm run web
```

### Running the Backend

```bash
# Install dependencies
cd apps/backend
pip install -r requirements.txt

# Start the server
python main.py

# Or with uvicorn for development
uvicorn main:app --reload --port 8000
```

### Running the Web Dashboard

```bash
# Install dependencies
cd apps/web-dashboard
npm install

# Start development server
npm run dev
```

## API Overview

The backend provides 129 RESTful endpoints across these categories:

| Category | Endpoints | Description |
|----------|-----------|-------------|
| Voice & Parsing | 8 | Voice command parsing, workout logging |
| AI Coach | 5 | Chat, exercise swaps, coaching |
| Programs | 5 | Generate strength, running, hybrid programs |
| Analytics | 10 | Volume, fatigue, deload analysis |
| Injuries | 6 | Detection, logging, recovery tracking |
| Wearables | 14 | OAuth, webhooks, metrics |
| Calendar | 5 | Scheduling, conflicts, availability |
| Badges | 6 | Achievement system |
| Health | 5 | Snapshots, correlations, predictions |

### Example API Calls

**Parse Voice Command**
```bash
curl -X POST http://localhost:8000/api/voice/parse \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"text": "bench press 225 for 8", "user_id": "uuid"}'
```

**Generate Training Program**
```bash
curl -X POST http://localhost:8000/api/program/generate/strength \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "uuid",
    "experience_level": "intermediate",
    "training_goals": ["hypertrophy", "strength"],
    "available_equipment": ["barbell", "dumbbells", "cables"],
    "training_frequency": 4
  }'
```

**Ask AI Coach**
```bash
curl -X POST http://localhost:8000/api/coach/ask \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"question": "Should I deload this week?", "user_id": "uuid"}'
```
